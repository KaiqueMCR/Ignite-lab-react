## ‚ñ∂Ô∏è Getting started

<br>

‚ö†Ô∏è to run this project you must have installed [Git](https://git-scm.com) and [Npm](https://nodejs.org/en/) or [Yarn](https://yarnpkg.com/).

<br>

1. Clone this repo with:

```bash
$ git clone https://github.com/KaiqueMCR/NLW-return-impulse-server.git
```

2. Install all dependences with:

```bash
$ yarn
```

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; or

```bash
$ npm install
```

3. Run dev server with:

```bash
$ yarn dev
```

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; or

```bash
$ npm run dev
```
